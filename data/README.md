# Dataset Information

This project uses the publicly available financial news dataset:

- Combined News DJIA Dataset
- Source: Kaggle
- Original Purpose:
  Predicting Dow Jones Industrial Average (DJIA) movement
  using daily financial news headlines.

## Dataset Structure

Main columns:

- Date
- Label
    - 0: DJIA Down
    - 1: DJIA Up
- Top1 ~ Top25
    - Daily financial news headlines

## Notes

- Raw dataset files are stored under the `data/` directory.
- Large files may not be displayed directly on GitHub preview.
- Preprocessed outputs are generated automatically through the preprocessing pipeline.
